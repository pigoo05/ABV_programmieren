
# Dokumentation der KI-Nutzung

Im Rahmen dieses Semesterprojekts wurde ChatGPT unterstuetzend eingesetzt.

## Verwendetes KI-Tool

- ChatGPT von OpenAI

## Einsatzbereiche

ChatGPT wurde fuer folgende Aufgaben verwendet:

- Strukturierung des Projekts
- Erklaerung der Anforderungen aus der Projektbeschreibung
- Unterstuetzung beim Aufbau von Frontend, Backend und Datenbank
- Hilfe beim Schreiben einfacher Code-Beispiele
- Hilfe beim Debugging von Fehlermeldungen
- Formulierungshilfe fuer README.md und diese Dokumentation

## Umfang der Unterstuetzung

Die Anwendung wurde schrittweise aufgebaut und lokal getestet. Die KI-Unterstuetzung wurde genutzt, um einzelne Schritte besser zu verstehen und einfache Loesungsansaetze zu entwickeln.

Die wichtigsten Entscheidungen zur Umsetzung waren:

- Verwendung von FastAPI fuer das Backend
- Verwendung von Streamlit fuer das Frontend
- Verwendung von SQLite als einfache relationale Datenbank
- Verwendung von Pydantic fuer die Datenvalidierung
- Verwendung von Pandas fuer die GuV-Auswertung
- Umsetzung eines Mini-Dashboards als kreative Erweiterung

## Eigene Bearbeitung

Der Code wurde schrittweise im eigenen Projektordner eingefuegt, ausgefuehrt, getestet und bei Fehlern angepasst. Die Funktionsweise der wichtigsten Bestandteile wurde nachvollzogen:

- Speichern von Buchungen ueber die API
- Abrufen gespeicherter Buchungen
- Berechnung von Einnahmen, Ausgaben und Gewinn
- Darstellung der Ergebnisse im Streamlit-Frontend
- Auswertung der Ausgaben nach Kategorien

## Wesentlich durch KI unterstuetzte Teile

Folgende Teile wurden wesentlich mit Unterstuetzung von ChatGPT erstellt:

- Grundstruktur des Projekts
- einfache FastAPI-Endpunkte
- einfache SQLite-Anbindung
- Streamlit-Frontend
- README.md
- AI_USAGE.md

## Transparenzhinweis

Die Nutzung von KI erfolgte als Lern- und Arbeitsunterstuetzung. Die Anwendung wurde lokal ausgefuehrt und getestet.
